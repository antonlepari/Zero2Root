# GTFOBins

This directory contains notes and examples for Linux binaries that can be abused to bypass local security restrictions.

Each file focuses on a single binary and includes:

- Description
- Enumeration
- Shell Escape
- Sudo Abuse
- SUID Abuse
- File Read
- File Write
- File Upload
- File Download
- Library Load
- Practical Examples
- References

## Available Binaries

- awk
- bash
- cp
- curl
- find
- less
- nano
- python
- rsync
- socat
- tar
- vim
- wget
- ...

> Reference: https://gtfobins.org/
