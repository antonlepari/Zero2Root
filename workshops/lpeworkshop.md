# Local Privilege Escalation Workshop

A collection of workshop materials covering Local Privilege Escalation (LPE) techniques for both Linux and Windows.

## Repository

https://github.com/sagishahar/lpeworkshop

## Overview

The workshop includes:

- Linux Local Privilege Escalation
- Windows Local Privilege Escalation
- Attack Tree Methodology
- Hands-on Lab Exercises
- Exercise Walkthroughs
- Workshop Slides
- Windows Lab Setup Script

## Included Materials

- Workshop Slide Deck
- Linux Exercise Walkthrough
- Windows Exercise Walkthrough
- Linux Lab VM
- Kali VM
- Windows Exercise Videos
- Windows Tools
- Windows Lab Setup Guide

## Why It's Useful

This repository is an excellent learning resource for:

- Beginners learning privilege escalation
- Preparing for OSCP, PNPT, CPTS, CRTO, and similar certifications
- Practicing in lab environments
- Understanding common privilege escalation vectors
- Building a structured methodology

## Notes

The workshop is intended for educational purposes and should only be used in authorized lab environments.
